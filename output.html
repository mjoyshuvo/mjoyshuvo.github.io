Authorization Token  ¶

Create a New Token
POST/token

You may create your own authorization token using this action. It takes a JSON object containing email address and password.

Input Fields type and description:

  • email* (string) - Registered Email Address.

Output Fields type and description:

  • token (string) - Authorization Token. Use this token every time to access other API

  • user (object) - User Data.

    • id (integer) - User ID.
    • first_name (string) - First Name.
    • last_name (string) - Last Name.
    • email (string) - Email Address.
    • phone (string) - Mobile Number.
    • is_active (boolean) - If user is Active or not.
    • type (integer) - Type of User.
    • is_agent (boolean) - If user is an Agent or not.
    • is_admin (boolean) - If user is an Admin or not.
